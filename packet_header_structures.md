################################################################################
# 1. Ethernet II (14 bytes)
################################################################################

 0               16              32             48             64
 +------------------------------+------------------------------+
 |        Destination MAC       |        Source MAC            |
 +------------------------------+------------------------------+
 |           EtherType          |   (e.g., 0x0800 IPv4)        |
 +------------------------------+------------------------------+

- Destination MAC(48bit): 목적지 MAC 주소  
  예시 → ff:ff:ff:ff:ff:ff (브로드캐스트)
- Source MAC(48bit): 출발지 MAC 주소  
  예시 → 08:00:27:12:34:56
- EtherType(16bit): 상위 프로토콜 지정  
  예시 → 0x0800(IPv4), 0x0806(ARP)

# 2. ARP (28 bytes)
################################################################################

 0               16              32
 +----------------+---------------+-----------------------------+
 | Hardware Type  | Protocol Type | Hardware Size | Proto Size |
 +----------------+---------------+-----------------------------+
 |                 Operation      |                             |
 +--------------------------------------------------------------+
 |                     Sender MAC (48bit)                       |
 +--------------------------------------------------------------+
 |                     Sender IP  (32bit)                       |
 +--------------------------------------------------------------+
 |                     Target MAC (48bit)                       |
 +--------------------------------------------------------------+
 |                     Target IP  (32bit)                       |
 +--------------------------------------------------------------+

- Hardware Type: 1=Ethernet  
  예시 → 1
- Protocol Type: 0x0800=IPv4  
  예시 → 0x0800
- Hardware Size: MAC(6)  
  예시 → 6
- Protocol Size: IPv4(4)  
  예시 → 4
- Operation: 1=Request, 2=Reply  
  예시 → 1(ARP 요청)
- Sender MAC/IP  
  예시 → 08:00:27:AA:BB:CC / 192.168.0.10
- Target MAC/IP  
  예시 → 00:00:00:00:00:00 / 192.168.0.1 (미알 때)

# 3. IPv4 Header (20 bytes)
################################################################################

 0        4        8        12       16               32
 +--------+--------+--------+--------+-----------------+
 |Version |  IHL   |  DSCP  |  ECN   | Total Length    |
 +--------+--------+--------+--------+-----------------+
 |        Identification     |Flags| Fragment Offset   |
 +---------------------------+-----+-------------------+
 |   TTL   | Protocol | Header Checksum               |
 +---------+----------+--------------------------------+
 |                 Source IP Address                  |
 +----------------------------------------------------+
 |              Destination IP Address                |
 +----------------------------------------------------+
 |                     Options...                     |
 +----------------------------------------------------+

- Version: IPv4=4  
  예시 → 4
- IHL: 헤더 길이(기본 20B)  
  예시 → 5
- DSCP: QoS  
  예시 → 0 (Best Effort)
- ECN: 혼잡 알림  
  예시 → 0
- Total Length: 전체 패킷  
  예시 → 60
- Identification: 단편화 ID  
  예시 → 0x1c46
- Flags: DF/MF  
  예시 → DF=1 (단편화 금지)
- Fragment Offset  
  예시 → 0
- TTL  
  예시 → 64
- Protocol  
  예시 → 6(TCP) / 17(UDP)
- Checksum  
  예시 → 0xb861
- Source/Destination IP  
  예시 → 192.168.0.10 → 142.250.206.78(Google)

# 5. TCP Header (20 bytes)
################################################################################

 0                        16                       32
 +------------------------+-------------------------+
 |      Source Port       |     Destination Port    |
 +------------------------+-------------------------+
 |                 Sequence Number                  |
 +--------------------------------------------------+
 |             Acknowledgment Number                |
 +--------+------+----------------------------------+
 | DataOff| Resv |            Flags                 |
 +--------+------+----------------------------------+
 |               Window Size                        |
 +--------------------------------------------------+
 |        Checksum         |     Urgent Pointer     |
 +--------------------------------------------------+
 |                Options (if any)                  |
 +--------------------------------------------------+

- Source/Destination Port  
  예시 → 53622 → 443(HTTPS)
- Sequence Number  
  예시 → 12345678
- ACK Number  
  예시 → 87654321
- Data Offset  
  예시 → 5 (20B)
- Flags  
  예시 → SYN=1 / ACK=0
- Window Size  
  예시 → 64240
- Checksum  
  예시 → 0xd213
- Urgent Pointer  
  예시 → 0
