# ARP Header Structure (28 bytes)

```text
################################################################################
                              ARP Header (28 bytes)
################################################################################
┌──────────────┬──────────────┬──────────────┬──────────────┐
│ HardwareType │ ProtocolType │ HardwareSize │ ProtocolSize │
│  (2 bytes)   │   (2 bytes)  │   (1 byte)   │   (1 byte)   │
└──────────────┴──────────────┴──────────────┴──────────────┘
┌──────────────────────────────────────────────────────────────────────────────┐
│                              Operation (2 bytes)                             │
└──────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────┐
│                           Sender MAC Address (6 bytes)                       │
└──────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────┐
│                            Sender IP Address (4 bytes)                       │
└──────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────┐
│                           Target MAC Address (6 bytes)                       │
└──────────────────────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────────────────────┐
│                            Target IP Address (4 bytes)                       │
└──────────────────────────────────────────────────────────────────────────────┘

HardwareType (2 bytes)
 : 사용 네트워크 하드웨어 타입
   예시 → 1 (Ethernet)

ProtocolType (2 bytes)
 : 상위 계층 프로토콜 식별
   예시 → 0x0800 (IPv4)

HardwareSize (1 byte)
 : MAC 주소 길이
   예시 → 6

ProtocolSize (1 byte)
 : IP 주소 길이
   예시 → 4

Operation (2 bytes)
 : ARP 동작 구분
   예시 → 1 (ARP Request)
           2 (ARP Reply)

Sender MAC / IP
 : 송신자 주소 정보
   예시 → 08:00:27:AA:BB:CC / 192.168.0.10

Target MAC / IP
 : 목적지 주소 정보
   예시 → 00:00:00:00:00:00 / 192.168.0.1
