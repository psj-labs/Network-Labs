# MPLS Header Structure (4 bytes)

```text
################################################################################
                               MPLS Header (4 bytes)
################################################################################

┌─────────────────────────────┬─────────────┬───────┬────────────────┐
│            Label            │      TC     │   S   │      TTL       │
│          (20 bits)          │   (3 bits)  │(1 bit)│    (8 bits)    │
└─────────────────────────────┴─────────────┴───────┴────────────────┘


Label (20 bits)
 : 목적지까지 전달될 **라벨 기반 스위칭 경로(LSP) 식별자**
   예시 → 299792

TC – Traffic Class (3 bits)   (구 EXP 필드)
 : 패킷 우선순위 / QoS(PS/CoS) / ECN 표시용
   예시 → 0b000

S – Bottom of Stack (1 bit)
 : 라벨 스택의 마지막 라벨 표시
   - 1 → 해당 라벨이 최하단(끝)
   - 0 → 뒤에 라벨이 더 있음
   예시 → 1

TTL (8 bits)
 : 홉마다 감소하여 루프 방지
   예시 → 64
