# EGP(Exterior Gateway Protocol)
#### EGP is a routing protocol used between Autonomous Systems (AS) managed by different networks, allowing routing information to be exchanged between ASes and enabling paths to be selected based on policy.

--- 

## 한국어 원문
#### EGP는 서로 다른 네트워크가 관리하는 자율 시스템(AS) 간에서 사용되는 라우팅 프로토콜로, 각 AS 사이에서 라우팅 정보를 교환하고 정책에 따라 경로를 선택할 수 있도록 해줍니다.
